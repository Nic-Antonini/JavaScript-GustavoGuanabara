Clientes: A pessoa que está usando o sistema, que envia dados ou pede dados de um servidor (usuário).

Servidor: A parte que cuida dos dados, envia para o banco de dados, pega dados do banco e envia para o lado do cliente (front).

- JavaScript é uma linguagem Client Side: Por mais que essa linguagem sirva tanto para o back-end quanto front-end ele é focado e mais utilizado na parte do cliente, ou seja, no front-end.
